# mpitest
Test the HPC with the MPI codes named hello_world.c and hello_world.f .
## hello_world.c
It's a MPI-c code.
### helloc.script
It's a LSF script to run the compiled hello_world.c .
## hello_world.f
It's a MPI-fortran code.
### hellof.script
It's a LSF script to run the compiled hello_world.f .
